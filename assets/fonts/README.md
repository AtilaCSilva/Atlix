# Fontes locais (self-host)

Coloque aqui os arquivos `.woff2` com estes nomes exatos:

| Arquivo | Família | Peso / estilo |
|---|---|---|
| `inter-300.woff2` | Inter | 300 |
| `inter-400.woff2` | Inter | 400 |
| `inter-500.woff2` | Inter | 500 |
| `inter-600.woff2` | Inter | 600 |
| `playfair-display-400.woff2` | Playfair Display | 400 |
| `playfair-display-600.woff2` | Playfair Display | 600 |
| `playfair-display-400-italic.woff2` | Playfair Display | 400 italic |
| `fira-code-400.woff2` | Fira Code | 400 |
| `fira-code-500.woff2` | Fira Code | 500 |

Subconjunto recomendado: **latin** (ou latin-ext se precisar de acentos extras).
